# Projects

Project index and maintenance hub for Dongwoo Lee.

Live site: https://dev-heps.github.io/projects/

This repository is not meant to replace individual project repositories. Instead, it keeps a stable public index of projects and links each project to:

- source repository
- live demo or GitHub Pages site
- project write-up
- status and domain

## Updating Projects

Edit `docs/projects.js` and add or update a project entry.

Each project repository should keep its own README and, when useful, a GitHub Pages page under that repository. Use `templates/project-page.md` as the starting structure.

## Public Pages

- `docs/index.html`: project index
- `docs/guide/index.html`: maintenance guide
- `docs/projects.js`: project metadata
